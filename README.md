# 基于 nuxt 的个人博客系统 `客户端`

---

## 技术栈

vue+nuxt+less+es6+webpack

## 版本

`v1.0.0`

## 更新记录

`暂无更新`

## 完成的功能

1. 文章列表，文章详情
2. 分类列表，分类详情
3. 标签列表，标签详情
4. 项目列表，项目详情
5. 搜索列表，搜索详情

### 待完成的功能

1. 评论管理
2. github 登录
3. 获取用户位置
4. ...

## 注意事项

1. axios 需要写入 nuxt.config.js 中

```js
plugins: [{ src: '~/plugins/axios' }];
```

## 如何使用

1. npm install
2. npm run dev
